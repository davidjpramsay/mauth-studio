Start here for app code, architecture, schemas, docs, tests, repository maintenance, CI, and release work.

Required context:

1. Read `AGENTS.md`.
2. Read `docs/local-ai-workflow.md`.
3. Read the specific docs/tests for the subsystem being changed.

5. Run targeted tests before handing work back.
