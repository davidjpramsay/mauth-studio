# Mauth Studio 0.1.1

Mauth Studio 0.1.1 is the first downloadable updater-enabled alpha release of the standalone macOS mathematics assessment authoring app.

## Download

- macOS on Apple Silicon (`arm64`)
- Developer ID signed and notarized by Apple
- Distributed as `Mauth-Studio-0.1.1-arm64.dmg`

Open the DMG, move **Mauth Studio** to Applications, then launch it from Finder, Spotlight, or the Dock. Python, Node.js, a repository checkout, and open Terminal windows are not required for normal teacher use.

Users of version 0.1.0 must install 0.1.1 manually once because 0.1.0 predates the updater. Later updater-enabled releases can be installed through **Mauth Studio > Check for Updates...**. Mauth asks before downloading an update and asks again before restarting to install it.

## Included

- School test, exam booklet, worksheet, and mathematics-notes templates
- Questions, parts, subparts, choices, tables, working spaces, and page controls
- Student and manually editable Solutions modes
- MathJax mathematics plus JSXGraph, Penrose, Plotly, vector, 3D, network, Venn, and image diagram surfaces
- Visible local document files, autosave recovery, versions, reusable logos, external-folder selection, and ZIP backup/import
- Optional authenticated local bridge for Codex, Claude Code, Cursor, and MCP clients
- Blank new tests, exams, and worksheets that remain blank after reopening
- Teacher-controlled update checks through the signed GitHub alpha release channel

## Release Safety

- Mauth does not download or install updates without teacher confirmation
- The updater is enabled only in packaged releases with signed update metadata
- Development builds do not check for updates
- The release is published only after its DMG, ZIP, updater metadata, and blockmaps have all uploaded and passed verification

## Alpha Limitations

- Apple Silicon only; Intel and universal builds are not currently supplied
- This is early software. Use **Backup ZIP** regularly and keep important assessment files in a backed-up folder
- A separate clean-Mac download/install verification remains recommended before wider deployment
- Codex and Claude bridge helper commands currently require a local repository checkout and Node dependencies; the standalone app itself does not

## Development

Use `pnpm macos:dev` for source iteration. `pnpm macos:release` and `pnpm macos:ship` are reserved for traceable, versioned external releases.
