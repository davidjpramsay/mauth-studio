What is Mauth Studio?

Mauth Studio is a work-in-progress tool for teachers who make maths assessments. The app shows printable A4 pages in the browser. Your AI helper can help create tests, fix layout, improve diagrams, or change the app itself.

teacher reviews in browser -> AI helper works in the local repo

For teachers

Build printable maths tests, exams, and worksheets with title pages, questions, diagrams, tables, working space, and solutions.

  • Preview before print: see the A4 pages in the browser.
  • Keep files local: project files, autosave, logos, and versions stay on your machine.
  • Ask for help: Codex or Claude can help write, edit, check, and improve assessments.

For AI helpers

Use two chats if you can. This keeps app changes separate from test-writing work.

  • Mauth Development: change app code, tests, docs, and setup.
  • Mauth Authoring: make or polish tests using the running app.
  • Use the app: do not edit saved project files directly.

Still in development

Mauth Studio is alpha software. It is public so people and their agents can inspect it, fork it, clone it, and improve it.

  • Fork: make your own public copy on GitHub.
  • Clone: keep a local copy on your computer.
  • Expect changes: the app, docs, and agent workflow will keep improving.

What your agent can change

Your agent is allowed to work on the app, not just use it.

  • App code: React, FastAPI, shared types, tests, and docs.
  • Assessment content: questions, diagrams, formatting, and solutions.
  • Setup: install problems, checks, local servers, and GitHub Pages docs.
