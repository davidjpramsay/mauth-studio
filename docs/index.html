What is Mauth Studio?

Mauth Studio is a work-in-progress tool for teachers who make maths assessments. The app shows printable A4 pages in the same window where you edit questions, diagrams, tables, and solutions. Your AI helper can create tests, fix layout, improve diagrams, or change the app itself through the local bridge.

teacher reviews in Mauth -> AI helper uses the structured local bridge

For teachers

Build printable maths tests, exams, and worksheets with title pages, questions, diagrams, tables, working space, and solutions.

  • Preview before print: see the A4 pages directly in Mauth Studio.
  • Keep files local: choose where your visible documents live, with autosave, logos, and versions on your Mac.
  • Ask for help: Codex or Claude can help write, edit, check, and improve assessments.

For AI helpers

Use two chats if you can. This keeps app changes separate from test-writing work.

  • Mauth Development: change app code, tests, docs, and setup.
  • Mauth Authoring: make or polish tests using the running app.
  • Use the app: do not edit saved project files directly.

Still in development

Mauth Studio is alpha software. The signed Apple Silicon app is ready for early use, while the public source remains open for inspection and improvement.

  • Download: use the signed and Apple-notarized Mac app.
  • Clone: connect local agent tools or develop the app.
  • Expect changes: the app, docs, and agent workflow will keep improving.

What your agent can change

Your agent is allowed to work on the app, not just use it.

  • App code: React, FastAPI, shared types, tests, and docs.
  • Assessment content: questions, diagrams, formatting, and solutions.
  • Setup: install problems, checks, local servers, and GitHub Pages docs.

Install

Open the downloaded DMG, move Mauth Studio to Applications, then launch it from Finder, Spotlight, or the Dock.

  • Platform: macOS on Apple Silicon.
  • Trust: Developer ID signed, notarized, and Gatekeeper-validated.
  • Updates: manual during the alpha; download a newer DMG when a release is published.
  • Stage: alpha software intended for early testing and feedback.

Connect Codex or Claude.

The downloaded app works by itself. For structured AI authoring, copy this prompt into a new Codex or Claude Code chat. It asks the agent to install Mauth, connect the repository helper tools, and verify the private local bridge.
